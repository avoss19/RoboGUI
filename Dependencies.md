# Mac OS X

- Homebrew
```
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

- qt4
```
$ brew install qt@4
```

- PyGame
```
$ pip install pygame
or
$ python -m pip install pygame
```
